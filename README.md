# Neftegorsk — Экономическая стратегия про заправки

Игра для Android на Godot 4. Игрок управляет сетью заправок в условиях дефицита топлива, выкупая конкуентов и развивая бизнес.

## 🎮 Концепция

**Сеттинг:** Город в условиях дефицита топлива (пост-кризис, логистический коллапс или санкции).  
**Роль игрока:** Владелец независимой заправочной сети.  
**Цель:** Стать монополистом во всех районах города, выкупив все чужые заправки.  
**Мета-игра:** Прогрессия через районы (как карта в Homescapes), звёзды за уровни → улучшения.

## 🏙️ Структура игры

### Карта города (Главное меню)
- Стиль: Санкт-Петербург (Нева, острова, мосты, исторический центр)
- 10 районов, открывающихся последовательно
- Иконки районов на карте → клик → выбор уровня

### Уровень (Геймплей)
- Процедурная генерация карты под район (сетка 6×6 – 10×10)
- Дорожная сеть, здания, вода, парки
- Игрок ставит 1 заправку → развивает
- Противники (1-4) с архетипами ИИ
- Экономика: цены, спрос, закупки, выкуп станций
- Победа: все станции противников выкуплены

### Архетипы противников (6 шт.)
1. **Акула** — агрессивная, демпингует, экспандирует
2. **Скупой** — держится за свои, высокие цены, лоялен
3. **Опортунист** — копирует игрока, адаптивен
4. **Магнат** (позже) — богат, рискует, саботирует
5. **Местный** — защищает территорию, кластеры станций
6. **Эколог** — премиум "зелёное" топливо, ЭЗ

### Улучшения (Дерево как в Homescapes)
6 веток × 5 тиров = 30+ улучшений:
- **Заправка** — баки, колонки, мульти-колонки, ЭЗ
- **Логистика** — автозаказ, оптовик, свои цистерны
- **Маркетинг** — реклама, лояльность, динамические цены, ИИ
- **Персонал** — обучение, менеджер, региональный директор
- **Технологии** — аналитика, прогнозы, баттехи
- **Особое** — франчайзинг, лицензия монополиста

## 🛠 Технологический стек

- **Engine:** Godot 4.2+
- **Language:** GDScript
- **Target:** Android (API 21+), порт на PC/веб
- **Architecture:** Autoload managers + сцены
- **Data:** Resource-based (`.tres` для районов, уровней, улучшений)
- **Save:** ConfigFile (JSON-совместимый) в `user://`


## 🚀 Запуск

1. Установите Godot 4.2+
2. Откройте `project/` как проект
3. Запустите сцену `scenes/main_menu/main_menu.tscn`
4. Для Android: Export → Android Preset → Build

## ⚙️ Настройка баланса

Все игровые параметры в Resource-файлах (`scripts/resources/`):
- `DistrictData` — плотность, трафик, цены земли, пул противников
- `OpponentArchetype` — черты характера, экономическое поведение
- `LevelData` — стартовый капитал, цели, бонусные звёзды
- `UpgradeData` — стоимость в звёздах/деньгах, эффекты, пререквизиты
